# Screenshots folder

This folder should contain screenshots of the SmartHub project for use on LinkedIn and in documentation.

Follow the guide in `../SCREENSHOTS.md` to capture these images:

3. `postman-collection.png` - Postman API tests
4. `database-schema.png` - Database tables and relationships

6. `code-example.png` - Well-documented code sample
7. `tests-passing.png` - Test results screenshot
8. `github-actions-ci.png` - GitHub Actions workflow success
9. `api-response-example.png` - API request/response in terminal

## Optional screenshots

- `architecture-diagram.png` - Custom architecture visualization
- `rate-limiting-demo.png` - Rate limiting in action
- `jwt-token-decoded.png` - JWT token structure on jwt.io
- `health-check.png` - Health check endpoint response

## Image guidelines

- Use PNG format for sharp text
- Minimum resolution: 1920x1080
- Keep file size reasonably small (compress if needed)
- Remove personal information (real emails, passwords, connection strings)
- Use demo data for examples
- Use consistent naming (kebab-case) and descriptive filenames

## Usage

These screenshots may be used in:
- LinkedIn Featured section
- LinkedIn carousel post
- GitHub README.md
- Portfolio website
- Resume/CV
