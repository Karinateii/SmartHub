## Image Guidelines

### Quality
- Use PNG format for sharp text
- Minimum resolution: 1920x1080
- Keep file size under 500KB each (compress if needed)

### Content
- Remove personal information (real emails, passwords)
- Use demo data (demo@example.com, John Doe)
- Crop unnecessary UI elements
- Use consistent theme (dark mode recommended)

### Naming Convention
- Use kebab-case: `swagger-overview.png`
- Be descriptive: `github-actions-ci-passing.png`
- Avoid spaces in filenames

## Tools for Compression

If your images are too large:
- [TinyPNG](https://tinypng.com/) - Online compression
- [ImageOptim](https://imageoptim.com/) - Mac app
- [PNGGauntlet](https://pnggauntlet.com/) - Windows app
